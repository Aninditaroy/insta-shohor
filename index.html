Javascript is a high-level programming language.Basically when the web browser loads a web page,it parses the HTML, and creates DOM (Document Object Model) from the contents. The DOM presents a live site view of the web page to our JavaScript code.Javascript works from top to bottom.We write the code and run our code in browser that has 'V8' javascript machine which interpreted and gets excuted each code line by line.

JavaScript has a event listeners and event loop based on runtime model.It is responsible for execute the code,handles the event loop queues,which is the secret that works behind javascript's programming.Example: setTimeout

Local storage data stored in our browser until explicitly deleted.Chnages can also be saved and modified for all current and future visits to the site.

Session storage data can be saved or deleted until the tab is closed.It changes per tab.Once the tab is closed, all the session data is deleted.
